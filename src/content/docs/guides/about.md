[Qozy Cue](https://qozycue.com/) adalah aplikasi berbasis online untuk membantu para pengusaha billiard memantau aktifitas bisnisnya secara real-time -- dimanapun mereka berada.

Tujuan pembuatan aplikasi ini awalnya sekedar penambah portfolio pribadi semata sebagai seorang [Software Engineer](https://github.com/alfieqashwa), namun kemudian dikembangkan secara serius ketika mendapati betapa amat sedikit -- jika enggan untuk mengatakan tidak ada -- aplikasi khusus billiard yang berbasis online.

## Visi

Baik aplikasi berbasis offline maupun online masing-masing memiliki kelebihan dan keterbatasannya. Tujuan dikembangkan [Qozy Cue](https://qozycue.com) untuk memberikan alternatif yang tidak didapati oleh aplikasi berbasis offline, diantaranya:

- Memanfaatkan teknologi internet agar para pemilik dapat memantau aktifitas bisnis mereka secara real-time dimanapun mereka berada.
- Mengurangi biaya seperti tidak diperlukan pembelian server serta perawatannya.
- Memungkinkan pengembang aplikasi untuk melakukan maintentance serta memberikan service dengan sangat efektif dan efisien.
- Menciptakan _evolving application_ (aplikasi yang terus diperbarui teknologinya seiring waktu serta menambah fitur-fitur baru).
